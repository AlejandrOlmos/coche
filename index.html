APIs

Con la apariciòn de HTML5 el desarrollo de los dispositivos mòviles con mejores capacidades se han desarrollado de distintas herramientas que nos ayudan a explotar el funcinamiando de las aplicaciones con distintos recursos de hardware y software.

Sin embargo, la mayor desventaja de las APIs es el soporte que se da a estas nuevas tecnologìas por parte de los navegadores, que muchas veces suele ser lento o completamente nulo.

Algunas de las herramientas ùtiles que estàn orientadas a las tecnologìas mòviles son las APIs de: Vibraciòn, geolocalizaciòn almacenamiento local, funcionalidad OFFline, streaming en càmara, etc.

Almacenamiento local

Una de las caracterìsticas màs interesantes de las nuevas herramientas de desarrollo es la que nos permite almacenar informaciòn dentro del navegador. Cuando escribimos una aplicacioòn web utilizamos tecnologìas de almacenamiento de datos que conmute se guardan en un servior de Base de datos. Sin embargo, si la aplicacioòn se encuentra offline, estos datos que estàn guardados en algìn servidor en internet no se encuentran disponibles. Debido a esto, se desarrllò una tecnologìa de llamada LocalStorage y SessionStorage que permite almacenar infomraciòn de manera permanente en el navegador.

La sintaxis del almacenamiento local es bastante sencilla. Primeramente debemos acceder a la funciòn llamada localStorage, despuès utilizamos algùn mètodo para indicar la acciòn que queremos llevar a cabo sobre el almacenamiento local. Finalmente, la asignaciòn de un valor funciona con un emparejado a una llave, esto quiere decir, logueamos un valor a una llave o nombre de variable.

localStorage.setItem('nombre','Eduardo')
